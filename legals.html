Ce site est réalisé en langages HTML et CSS3, PHP, Laravel, Blade, MySQL. Pour une navigation fluide nous vous recommandons d’utiliser les dernières versions des navigateurs

Conformément à la loi « informatique et liberté » du 6 janvier 1978 modifiée en 2004, vous bénéficiez d’un droit d’accès, de rectification, de modification et de suppression concernant vos données personnelles collectées sur le site Yalla des enfants. COOKIES

Pour faciliter la navigation, un fichier texte est déposé sur votre ordinateur lors de l’arrivée sur le site. Ces données personnelles permet d’enregistrer le parcours de l'utilisateur et d’améliorer les fonctionnalités du services (par exemple, pour établir des statistiques de fréquentations). Les cookies mémoriseront la langue d’un site ou permettra de faire une publicité ciblée. Certaines pages demandent l’accord de l’utilisateur pour accepter les cookies. En cas de refus, la navigation sur le site sera limitée.


Vous pouvez désactiver les cookies ou modifier les paramètres dans le navigateur :

  • Google Chrome : https://support.google.com/chrome/answer/95647?hl=fr
  • Internet Explorer : http://windows.microsoft.com/fr-fr/internet-explorer/delete-manage-cookies#ie=ie-11
  • Mozilla Firefox : http://support.mozilla.org/fr/kb/activer-desactiver-cookies
  • Safari : http://www.apple.com/fr/privacy/use-of-cookies
